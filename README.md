# Henric_Caasi | Portfolio

Personal portfolio for Rogel Henric M. Caasi, built with Next.js and Tailwind CSS.

## Run Locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Deploy To Vercel

1. Create a new GitHub repository named `henric-caasi-portfolio`.
2. Push this folder to that repository.
3. In Vercel, choose `Add New Project`.
4. Import the GitHub repository.
5. Keep the default settings:
   - Framework Preset: `Next.js`
   - Build Command: `next build`
   - Output Directory: `.next`
6. Click `Deploy`.

## Update Content

- Resume: replace `public/assets/Henric_Caasi_Resume.pdf`.
- Project screenshots: replace files inside `public/assets`.
- Main page content: edit `app/page.jsx`.
